# Student Performance Prediction

##

Description

This project is a Machine Learning model that predicts student performance based on factors like study hours, attendance, and previous scores using Linear Regression.

##

Objective
- To analyze how different factors affect student performance
- To build a predictive model using Linear Regression
- To help improve academic performance analysis

## 🛠️ Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib

## 📊 Dataset
The dataset includes:
- Study Hours
- Attendance
- Previous Scores
- Final Marks (Target)

## 🔄 Workflow
1. Data Collection  
2. Data Preprocessing  
3. Train-Test Split  
4. Apply Linear Regression  
5. Model Training  
6. Prediction  
7. Result Evaluation## 📈 Output
The model predicts student marks based on input features.  
Higher study hours and attendance generally lead to better performance.

## 🧠 Conclusion
The project shows that student performance can be predicted using Machine Learning. Linear Regression provides a simple and effective way to understand the relationship between study habits and marks.
